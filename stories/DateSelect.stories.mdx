# Date Select

The Date Select is a simple component that allows the user to select a date. It is
a wrapper around the `Calendar` component and uses the `TextInput` component to
display the date.

## Functionality

The functionality for the Date Select component is fairly simple, all you need
to do is pass the props that you would want for the component and that's it.

This component particularly plays nice with the `Form` and is able to register it's
inputs when needed.
